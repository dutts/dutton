DataBinding works like this:

  • The binding path resolves.
  • The value converter, if specified can convert the value.
  • The resulting value is valid for the property you are binding the value to (the binding target, e.g. TextProperty on TextBlock).

If the either of the firts two steps result in a DependencyProperty.UnsetValue, the target proprety will be set to the value of FallbackValue.

FallbackValue
http://msdn.microsoft.com/en-us/library/system.windows.data.bindingbase.fallbackvalue.aspx

PriorityBinding
http://msdn.microsoft.com/en-us/library/system.windows.data.prioritybinding.aspx
NB: PriorityBinding will treat a null string as a successful binding.
